##Quick10KeyLauncher
A lightweight Windows launcher that assigns files, folders, and URLs to numeric keypad hotkeys.

## Screenshot
![Main Screen](https://github.com/manmosu-cyber/Quick10KeyLauncher/blob/main/ScreenShot_main.png)

##Features
Assign hotkeys to any file, folder, or URL
Designed specifically for numeric keypad usage
Very fast and lightweight
Minimal UI focused on speed
No background process when idle
Simple configuration using a text-based list
MIT licensed

##Installation
Download the installer from the Releases page:
Quick10KeyLauncher_Setup_V1.1.exe
The installer is built using Inno Setup.

##Usage
### Assigning items
You can assign files or folders to numpad keys in two simple ways:

**1. Drag & Drop**
Drag a file or folder onto a key button in the main window.  
The item is immediately assigned to that key.

**2. Edit the List folder**
Each numpad key has a corresponding text file in the `List` folder  
(e.g., `1.txt`, `2.txt`, `0.txt`).  
Write the full path of the item you want to launch.

### Launching
Press the corresponding numpad key to open the assigned item instantly.
